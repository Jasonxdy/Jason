문단을 나누는 태그(p, pre)

문단 영역을 나누는 태그로는 <p>와 <pre>가 있음.

<p>는 문단 영역을 나누는 태그이지만
연속된 공백문자 입력 시 하나의 공백만 표시하며
줄 바꿈 입력을 별도의 태그로 지정을 해야함. 

<pre>는 여러칸      띄어쓰기,
줄바꿈 혹은 자동 입력된 tab키 등 
입력한 내용을 그대로 표현하는 태그

그 밖의 텍스트를 다루는 태그들

- 일반글꼴
<b> : 단순히 글자를 굵게 표시하는 태그
<strong> : 글자를 강조하는 태그 (웹 접근성)
<i> : 단순히 글자를 기울이는 태그
<em> : 글자를 강조하는 태그 (웹 접근성)
<blockquote> : 인용문구를 나타내는 태그
- cite 속성 : 인용된 사이트 주소를 작성
직접 노출은 되지 않지만, 검색엔진에는 노출됨.

<mark>: 형광펜 효과를 나타내는 태그
<u> : 글자에 밑줄을 긋는 태그
<small> : 글자를 작게 표시하는 태그
<sup> : 윗첨자를 나타내는 태그
<sub> : 아랫첨자를 나타내는 태그
<s> : 글자에 취소선을 넣는 태그

<abbr> : 마우스 오버 시 툴팁 형태로 출력하게 만들어주는 태그
IoT란 각종 사물에 센서와 통신 기능을 내장해 인터넷에 연결하는 기능.


<kbd> : 키보드 같은 사용자 입력 내용을 표시만 해주는 태그 Ctrl + k

글자 관련 태그 응용

태그들은 해당 태그 내에서 중첩으로 사용 가능하다.
굵은글자를 사용할 수도 있고, 기울이거나,취소선을 넣는 등 다양하게 사용할 수 있다.
